## Netflutter

### Description

- 넷플릭스 UI를 모방해서 만든 Flutter 앱입니다.
- Firebase를 이용해 앱 데이터를 관리했습니다.
<br>

### Features

- 홈 화면 콘텐츠 슬라이드
- 콘텐츠 상세 페이지
- 콘텐츠 필터링 검색
- 찜 기능, 찜한 콘텐츠 목록
- 프로필 페이지, 프로필 수정
<br>

### How to run the project

```bash
# 개발 환경 진단
$ flutter doctor

# 의존성 패키지 설치
$ flutter pub get

# 앱 실행
$ flutter run
```
